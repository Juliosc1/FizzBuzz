# FizzBuzz
Skapa ett FizzBuzz
1. Om talet är delbart med 3 = Fizz
2. Om talet är delbart med 5 = Buzz
3. Om talet är delbart med 3 och 5 eller med 0 = FizzBuzz
4. Annars returnerar bara siffran
